# Darshit Portfolio

Professional portfolio website for Darshit Rajyaguru, built to present WordPress expertise, client projects, and contribution credibility in a polished and responsive static site.

## Overview

This portfolio highlights:

- WordPress expert positioning with core contributor credibility
- Custom WordPress development services and client-focused messaging
- Featured projects and detailed project case studies
- Contact and profile links for client inquiries
- Responsive layout with a mobile slide-in navigation menu

## Pages

- `index.html` - homepage, hero section, WordPress contribution trust section, featured work
- `about.html` - professional summary, expertise, and strengths
- `projects.html` - project showcase with popup details
- `contact.html` - direct contact links and professional profiles

## Design Improvements Included

- modern premium visual layout
- stronger personal branding and positioning
- better typography and spacing
- improved card and grid alignment
- responsive mobile and tablet navigation
- WordPress ecosystem trust section with earned contribution badges

## Tech
